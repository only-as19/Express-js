# Express.js Basics

This repository demonstrates the fundamental concepts of backend development using Express.js.

## Features
- Basic Express server setup
- Routing with GET and POST
- Request and Response handling
- Introduction to middleware

## Tech Stack
- Node.js
- Express.js

## How to Run the Project

1. Clone the repository
git clone https://github.com/only-as19/Express-js.git

2. Install dependencies
npm install

3. Run the server
node index.js
